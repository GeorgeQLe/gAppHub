Medium Badge legend lacks documented hover tooltips

The UI spec says each badge legend item should show a one-line tooltip on hover, but src/components/BadgeLegend.tsx only renders static labels. src/__tests__/BadgeLegend.test.tsx covers item presence but not tooltip behavior. Recommended fix: restore legend tooltip state and add a focused test for hover/focus disclosure.

Medium Page indicator can use unclamped active state

src/components/IconGrid.tsx clamps rendered content with safePage, but passes raw page to PageDots. When the row calculation changes the page count, no dot can be selected even though the content is clamped to a valid page. Recommended fix: pass safePage to PageDots and normalize page when totalPages shrinks.
