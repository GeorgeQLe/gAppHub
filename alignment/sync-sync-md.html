Approved sync.md preview

Post-sync actions for GappHub

This preview records the project-level automation that will run after future $sync pulls: dependency refresh, the test suite, and alerts for files that need human review when changed upstream.

Canonical File

The approved project file is sync.md at the repository root.

Commands

pnpm install
pnpm test

Watch List

  • package.json - check for dependency or script changes
  • pnpm-lock.yaml - refresh dependencies
  • AGENTS.md - review project conventions
  • tasks/todo.md - review active work queue
